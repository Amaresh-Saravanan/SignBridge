Integration Flow
----------------
1. User speaks or types text
2. handleHearingTranslation called
3. POST /api/translate sent to backend
4. Backend returns glosses array
5. Gloss tokens extracted and enqueued
6. HubAvatar dequeues and plays animations
7. NMM triggers applied for facial expressions

API Integration
---------------
Endpoint: POST http://localhost:3001/api/translate
Request: { "sentence": "I eat rice" }
Response: {
  "success": true,
  "data": {
    "glosses": [{ "token": "I" }, { "token": "EAT" }],
    "nmm": { "eyebrows": "neutral", "head": "neutral" }
  }
}

Error Handling
--------------
- API failure: Falls back to caption with "(translation failed)" suffix
- Empty response: Skips animation enqueue
- Network error: Caught and logged, signing not triggered
