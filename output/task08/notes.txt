Summary

Built the Express/Node.js backend with an ISL Translation API that converts

Files Created
-------------
1. backend/package.json
   - Express, CORS, Helmet, Rate Limiting, Zod dependencies

2. backend/server.js
   - Express server with security middleware
   - CORS configuration for frontend
   - Rate limiting (100 requests per 15 minutes)
   - Error handling middleware

3. backend/services/islTranslator.js
   - English tokenization and parsing
   - Auxiliary word stripping (is, am, are, the, etc.)
   - Tense detection (past, present continuous, future)
   - S-V-O to S-O-V rearrangement
   - Negation handling with NOT marker
   - Question type detection (WH, Yes/No)
   - NMM trigger generation

4. backend/routes/translate.js
   - POST /api/translate - Main translation endpoint
   - GET /api/glosses - Available gloss vocabulary
   - GET /api/health - Health check

5. backend/middleware/validate.js
   - Zod schema validation for translate input
   - Input sanitization and length limits

API Endpoints
-------------
POST /api/translate
  Body: { "sentence": "I eat rice", "context": "everyday" }
  Response: {
    "success": true,
    "data": {
      "sentence": "I eat rice",
      "glosses": [
        { "token": "I", "type": "subject", "nmm": null },
        { "token": "RICE", "type": "object", "nmm": null },
        { "token": "EAT", "type": "verb", "nmm": null }
      ],
      "nmm": { "eyebrows": "neutral", "head": "neutral", "gaze": "forward" },
      "metadata": { "tense": null, "questionType": null, "isNegation": false }
    }
  }

GET /api/glosses
  Response: { "success": true, "data": { "glosses": [...], "count": 85 } }

GET /api/health
  Response: { "status": "healthy", "service": "signbridge-translate" }

Translation Examples
--------------------
Input: "I eat rice"
Output: I RICE EAT (SOV)

Input: "I will go to school tomorrow"
Output: TOMORROW I SCHOOL GO (Tense + SOV)

Input: "Do you like coffee?"
Output: YOU COFFEE LIKE ? (NMM: eyebrows raised)

Input: "I don't like it"
Output: I LIKE NOT (Negation at end)
