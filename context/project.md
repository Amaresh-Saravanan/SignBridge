# Project Overview: SignBridge

## Description
**SignBridge** is an accessibility and translation platform designed to bridge the communication gap between the hearing and Deaf/Hard-of-Hearing (HoH) communities. The platform translates English text or speech in real-time into Indian Sign Language (ISL), represented by an animated, professional cartoon-style 3D avatar.

## Purpose
Convert English text and spoken input into grammatically correct Indian Sign Language using a real-time, WebGL-rendered 3D avatar.

## Target Users
- **Deaf & Hard-of-Hearing individuals**: Accessing real-time captions and sign translation in classrooms, public offices, or at home.
- **Educational institutions**: Teaching Indian Sign Language or delivering lectures with integrated avatar sign translations.
- **Government accessibility services**: Deploying sign language translations on digital portals, information hubs, and emergency announcements.

## Key System Features
1. **Live Translation Engine**: Intercepts speech input, translates sentences into ISL glosses, and drives character animations.
2. **Interactive 3D Player**: WebGL canvas that renders a fully rigged character playing signs smoothly with transitions.
3. **Admin Dictionary Portal**: A web interface for dictionary collectors to add, review, and adjust sign animations.
