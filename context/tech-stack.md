# Technology Stack Reference

## Frontend Environment
- **React**: Web interface layer, layout templates, state hooks, and side effects.
- **Vite**: Ultra-fast build and bundling configuration.
- **TypeScript**: Static typing for data structures, API requests, and configurations.
- **Tailwind CSS**: Styling, dark-mode styling, responsive layouts.

## 3D Render & Asset Optimization
- **Blender**: 3D character mesh generation, topology cleanups, rigging setups, keyframe animation clips, shape key visemes.
- **Three.js**: Lower-level WebGL graphics engine, scenes, cameras, materials, meshes, bones, and light sources.
- **React Three Fiber (R3F)**: Declarative wrapper for Three.js in React component trees.
- **Drei**: Helper hooks (`useGLTF`, `useAnimations`) and layout components for R3F.

## Backend Translation Service
- **Node.js & Express**: Backend API service hosting authentication, dictionary databases, translation scripts, and audio file processors.

## Animation & Asset Export
- **GLTF/GLB**: Standard delivery format for 3D meshes, armatures, materials, and custom shape keys.
- **AnimationMixer / GLTFLoader**: Three.js utilities used to play GLB animations and transition tracks.
